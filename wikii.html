QR Attendance Management System

trainee portal

creat a class session and generate a QR code for student to scan.

trainee requirements

  • register class session
  • generate qr code for attendance
  • ensure student scan upon arrival

student portal

scan the QR code when you arrive in class to mark attendance.

student details

  • full name
  • student id
  • class session

administrator portal

view attendance records submitted by students.

administrator requirements

  • monitor attendance records
  • generate attendance reports
  • manage trainee and student accounts
